# 单图片审核
## 项目简介
本项目使用 [NudeNet](https://github.com/notAI-tech/NudeNet) 库实现对单张图片的内容进行审核，判断图片是否含有不适宜的内容。

## 功能列表
1. 可选遮罩类型：色块、高斯模糊、马赛克，或只检测不处理。
2. 自定义色块颜色。
3. 可调节模糊强度。
4. 可选遮罩形状：矩形、椭圆。
5. 可缩放遮罩比例。
6. 边缘羽化效果，限：色块、高斯模糊。
7. 可选需要处理的标签。
8. 支持从 URL、Base64 编码字符串或文件加载图片。
